# 在windows下安装theano

# 0.说明

本教程记录了我在Windows下安装theano的过程

注意本教程中的所有命令根据你的实际情况而定，请不要原样粘贴。

如果你对本教程有任何问题或者指出错误，请联系我。

# 1.准备工作

* Windows 10 Pro X64
* Anaconda
* TDM-GCC X64


# 2.安装并更新Anaconda

	conda update conda
	
	conda update --all
	

# 3.安装TDM-GCC

# 4.安装相关依赖包

	conda install mingw libpython
	
# 5.安装theano

	pip install Theano
